ALGORITHM
---------
1. Load microsoft/codebert-base (transformers, AutoModel + AutoTokenizer).
2. For each class in truth(733) + unresolved(478) = 1211 total:
   - Get class_code from v20_hybrid_best.db classes table.
   - Tokenize truncate 512; if longer, chunk by 512 and mean-pool chunks.
   - Sentence vector = mean-pool last_hidden_state (attention-masked).
   - Result: 768-dim vector per class.
3. Upsert all vectors to Qdrant collection 'dom_class_code'
   (size=768, distance=Cosine; payload {class_name, owner, layer}).
   Qdrant is at localhost:6333 (default).
4. Per-domain CENTROID = mean of truth vectors for that owner,
   ONLY for owners with >= 3 truth classes (stable centroid).
5. For each unresolved class: cosine to every centroid, take max.
   - score >= THRESHOLD (0.60) -> INSERT into domain_hypothesis
       (class_name, owner, tier=3, method='codebert-knn', score)
       and DELETE that row from domain_unresolved.
   - score < THRESHOLD -> leave in domain_unresolved (do nothing).
